# Постановка проблемы

В моей учебной группе мы делили билеты между всеми для их росписи. После ещё и проверяли с целью исправить ошибки. Из-за большего количества участников и хаотичной росписи и проверки билеты перемешивались, из-за чего в итоговый файл могли попасть старые версии билетов с ошибками, которые уже ранее исправляли.

# Решение задачи

Для решения данной проблемы был создан телеграмм бот, в который люди загружали свои билеты, а он в ответ выдавал итоговый файл, который состоят из билетов последней версии. 

Другими словами, проблема с контролем версий и сборкой итогового файла была переложена на плечи тг-бота.
